#  Book Recommender System

A semantic book recommendation system using:
- Transformers
- ChromaDB
- Hugging Face models
- Scikit-learn baseline classifier
- Gradio UI

## Features
- Embeddings
- Emotion-based ranking
- Category prediction (Sklearn & HF Zero-Shot)
- Modular ML architecture
- Ready for RAG integration

## Tech Stack
- Python
- LangChain
- HuggingFace
- Scikit-learn
- ChromaDB
- Gradio

## How to Run
```bash
pip install -r requirements.txt
python app.py
